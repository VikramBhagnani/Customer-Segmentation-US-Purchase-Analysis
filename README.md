# Customer Segmentation & US Purchase Analysis

## Project Overview
This project analyzes customer purchasing behavior in a US retail dataset and applies machine learning techniques to segment customers based on spending patterns and purchasing activity.

The goal is to help businesses understand customer groups and design targeted marketing strategies.

## Business Objectives
• Analyze revenue trends across categories, seasons, and age groups  
• Identify high-value customer segments  
• Apply K-Means clustering to group customers based on behavior  
• Build an interactive Power BI dashboard for business insights  

## Tools & Technologies
Python  
Pandas  
Scikit-Learn  
Matplotlib / Seaborn  
Power BI  

## Project Workflow
1. Data Cleaning and Preparation
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Customer Segmentation using K-Means Clustering
5. Business Insights and Recommendations
6. Interactive Dashboard Development (Power BI)

## Key Insights
• Customers aged **46-60 generate the highest revenue**  
• **Clothing and Accessories** categories drive the largest sales  
• **Spring season** records the highest overall revenue  
• Cluster analysis identifies **high-value and low-value customer segments**
